# KA 02 - Software Architecture

Bem-vindo ao Knowledge Area de Arquitetura de Software do SWEBOK-AI v5.0.

Neste módulo, exploramos como a arquitetura de software evoluiu de uma
disciplina puramente estrutural para a gestão de contexto e sistemas híbridos na
era da Inteligência Artificial.
