# KA 02 - Arquitetura de Sistemas Híbridos: Visão Geral

## Escopo

Este KA define princípios, estruturas e limites para arquiteturas de sistemas
híbridos com componentes determinísticos e probabilísticos. O foco recai sobre
supervisão humana, mecanismos de controle, auditabilidade ponta a ponta,
separação de concerns críticos e atributos de qualidade arquitetural
(confiabilidade, segurança, observabilidade e evolutividade).

## Objetivos de Aprendizagem

1. Distinguir responsabilidades entre componentes determinísticos e
   probabilísticos em arquiteturas híbridas.
2. Definir estratégias de supervisão e controle para reduzir risco operacional e
   regulatório.
3. Projetar trilhas de auditabilidade e rastreabilidade para decisões e
   artefatos gerados por IA.
4. Avaliar trade-offs arquiteturais de qualidade em cenários com não
   determinismo.
