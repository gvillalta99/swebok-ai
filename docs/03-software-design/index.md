# KA 03 - Design de Sistemas Híbridos: Visão Geral

## Escopo

Este KA aborda princípios e padrões de design para sistemas híbridos com IA, com
foco na delimitação de responsabilidades entre componentes determinísticos e
probabilísticos, na definição de interfaces e contratos, na verificabilidade
operacional e na modernização segura de sistemas, com auditabilidade e
rastreabilidade de decisões.
