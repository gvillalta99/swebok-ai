# Plano do Capítulo 18: Fundamentos de Engenharia para Sistemas Cognitivos

## Visão Geral

O capítulo de Engineering Foundations no SWEBOK-AI v5.0 representa uma
reconfiguração crítica dos fundamentos de engenharia aplicados à era dos
sistemas cognitivos artificiais. Enquanto o SWEBOK v4.0 focava em fundamentos de
engenharia tradicionais (processos, design, medição, padrões, análise de causa
raiz), a versão 5.0 reconhece que a engenharia de software tornou-se uma
disciplina de **orquestração e governança de sistemas autônomos**.

Este capítulo estabelece os **Fundamentos de Engenharia para Sistemas
Cognitivos** — uma disciplina que conecta princípios de engenharia tradicional
às necessidades emergentes de projetar, medir e validar sistemas híbridos
humanos-IA. O engenheiro de software do futuro precisa dominar não apenas
processos de engenharia, mas técnicas para garantir que sistemas de IA operem
dentro de limites seguros, auditáveis e economicamente viáveis.

A transição proposta expande o escopo tradicional para incluir:

- **Engenharia de sistemas opacos:** Técnicas para projetar e manter sistemas
  cujos mecanismos internos são parcialmente inacessíveis
- **Métricas para sistemas não-determinísticos:** Medições apropriadas para
  comportamento probabilístico de LLMs
- **Design para verificabilidade:** Princípios de design que facilitam validação
  de código gerado por IA
- **Governança de engenharia:** Frameworks para responsabilidade e compliance em
  sistemas autônomos

## Relacionamento com Outros KAs

- **Software Requirements:** Processos de engenharia para elicitação de
  restrições
- **Software Architecture:** Design de sistemas híbridos verificáveis
- **Software Design:** Abstração e encapsulação aplicados a sistemas cognitivos
- **Software Testing:** Métodos empíricos para validação de sistemas
  não-determinísticos
- **Software Engineering Economics:** Medições e análise estatística de
  custo-benefício
- **Mathematical Foundations:** Fundamentos estatísticos para análise de
  sistemas de IA
- **Computing Foundations:** Aplicação de fundamentos de sistemas cognitivos
  artificiais
